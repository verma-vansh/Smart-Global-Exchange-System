# Smart Global Exchange System — Premium Currency Converter

A world-class, professional currency conversion application built with modern web technologies. This project showcases high-end UI/UX design principles, real-time data integration, and advanced functional features.

## 🚀 Core Features
- **Real-Time Exchange Rates:** Fetches live data from global markets.
- **Premium UI/UX:** Glassmorphism design, smooth animations, and a responsive dashboard layout.
- **Voice Output:** Integrated accessibility feature for reading conversion results.
- **Conversion History:** Tracks recent activities for quick reference.
- **Market Watch:** Searchable currency list with favorite options.
- **Theme Switcher:** Seamless transition between Dark and Light modes.
- **Financial Insights:** "Did You Know?" section providing educational finance facts.

## 🛠️ Technology Stack
- **Frontend:** React 19, TypeScript
- **Styling:** Tailwind CSS 4.0
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **API:** Open Exchange Rates API (Free Tier)

## 📂 Folder Structure
- `/src/App.tsx`: Main application logic and UI components.
- `/src/constants.ts`: Currency data, symbols, flags, and financial facts.
- `/src/index.css`: Global styles and Tailwind configuration.
- `/metadata.json`: Project metadata and permissions.

## 💡 Innovation
Unlike standard converters, this system focuses on the **Emotional Design** of the interface, using gradients and glassmorphism to create a "Premium Dashboard" feel inspired by modern fintech applications like Revolut or Robinhood.
